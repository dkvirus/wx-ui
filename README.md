# wx-ui （持续补充中）
提供微信小程序常用的第三方组件，如价格加减组件、商品评论星星数组件等。

- [x] 1. 价格加减组件；
- [x] 2. 商品评价星星组件；
- [x] 3. 从底部弹出模态框；
- [x] 4. 自定义省市区组件；


使用微信小程序原生的 Component 构造器编写的组件，使用时直接复制 components 目录下需要的组件即可，使用方法见官方文档：[自定义组件使用文档](https://mp.weixin.qq.com/debug/wxadoc/dev/framework/custom-component/)
